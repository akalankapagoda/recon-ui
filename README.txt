This is the Transaction Reconciliation front-end.


Pre-Requisites
-----------------

1. Node 16+
2. NPM 8+


How to Build
--------------

Navigate to the project base directory using terminal and run

    `npm run build'
	
How to Run
------------

1. Update the REACT_APP_RECON_API_BASE_URL property in the .env file with the back-end URL.
2. Navigate to the project base directory using the terminal and run

	`npm start`

3. Navigate to http://localhost:3000 (replace port as required)
